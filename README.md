# AMS Clinical Assistant™ v2.0

Single-file GitHub Pages application.

## Included
- Module 1: New Resident Assessment
- Module 2: Differential Diagnosis
- Module 3: Rational Antibiotic Use / 48–72 Hour Timeout
- Generated SBAR/documentation
- Copy, Share, and Print/Save PDF
- Mobile-responsive interface
